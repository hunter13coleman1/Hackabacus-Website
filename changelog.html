Changelog

What's New in Hackabacus

Every update, improvement, and new feature we ship for the Excel and PowerPoint plugin trusted by PE, banking, and finance teams.

v2.4.0 March 10, 2026 Feature

Model Audit Dashboard & Slide Layout Engine

  • Model Audit Dashboard — New panel that scans Excel workbooks for hardcoded values, broken references, and circular dependencies in real time.
  • PowerPoint Layout Engine v2 — Completely rewritten slide layout engine with automatic alignment grids, margin snapping, and multi-object distribute tools.
  • Conditional number formatting — Apply DARP, MOIC, and IRR-aware number formats with a single shortcut. Supports parenthetical negatives and basis-point notation.
  • Improved COM add-in startup time by 40% on Windows through lazy initialization of non-critical modules.
v2.3.1 January 22, 2026 Bug Fix

Stability Patch & Excel 365 Compatibility

  • Fixed a regression where cell style presets were not persisting across workbook save and reopen in Excel 365 (Build 16.0.17928).
  • Resolved intermittent COM disconnection on macOS when switching between Excel and PowerPoint rapidly.
  • Corrected border formatting inconsistencies in merged cell ranges when using the Quick Format panel.
  • Updated certificate signing for Windows installer to resolve SmartScreen warnings on fresh installs.
v2.3.0 November 14, 2025 Feature

PowerPoint Slide Automation & Team Style Sync

  • Slide Automation Workflows — Build repeatable slide generation pipelines that pull data from Excel models directly into PowerPoint placeholders, tables, and charts.
  • Team Style Sync — Share cell style presets, number formats, and color palettes across your team via a centralized configuration file.
  • New keyboard shortcut editor for customizing all Hackabacus commands in both Excel and PowerPoint.
  • Added support for Apple Silicon native execution on macOS — eliminates Rosetta dependency for M1/M2/M3 machines.
v2.2.0 August 29, 2025 Improvement

Enhanced Number Formatting & Cell Styling

  • Smart Number Formatting — Automatically detects column context (currency, percentage, multiple) and suggests the appropriate format with configurable decimal precision.
  • Overhauled the cell styling toolbar with live preview, recently-used styles, and one-click border presets for financial tables.
  • New alternating row shading tool with customizable color steps that respects print-safe palettes.
  • Performance: reduced memory footprint by 25% when processing workbooks with 50+ sheets.
v2.1.0 June 5, 2025 Feature

Quick Format Panel & Print Optimization

  • Quick Format Panel — A dockable sidebar in Excel for applying institutional-grade formatting to selected ranges: fonts, borders, number formats, and column widths in one action.
  • Print Optimizer — Automatically adjusts page breaks, headers, footers, and scaling to produce clean PDF output from complex models.
  • Added VLOOKUP and INDEX/MATCH audit highlighting to trace formula dependencies across sheets visually.
  • Improved PowerPoint table styling with automatic header detection and finance-standard gridline presets.
v2.0.0 March 18, 2025 Feature

Hackabacus 2.0 — Full Rewrite

  • Complete architecture rewrite — Rebuilt from the ground up as a modern COM add-in with native .NET 8 runtime for dramatically improved stability and performance.
  • PowerPoint support — Hackabacus now extends beyond Excel with a full suite of slide formatting, layout, and automation tools for PowerPoint.
  • New ribbon UI — Redesigned ribbon interface with contextual tabs, search-as-you-type command palette, and customizable quick-access toolbar.
  • Cross-platform installer for Windows 10/11 and macOS 13+. Silent deployment via MSI and PKG for IT teams.
  • Introduced per-seat team licensing with centralized admin portal for managing deployments across the firm.
