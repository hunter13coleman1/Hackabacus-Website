# Hackabacus Installer Downloads

Place your installer files in this directory:

- `HackabacusSetup.exe` — Windows installer
- `Hackabacus.dmg` — macOS installer

These files are served via Vercel rewrites:
- hackabacus.com/download/windows → downloads/HackabacusSetup.exe
- hackabacus.com/download/mac → downloads/Hackabacus.dmg

The download page is at hackabacus.com/download
