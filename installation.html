Installation Guide

Get Hackabacus Up and Running

Follow these steps to install the Hackabacus plugin for Excel and PowerPoint on Windows. The entire process takes less than ten minutes.

01

System Requirements

Before you begin, make sure your machine meets the minimum requirements.

Operating System Windows 10 or later (64-bit)
Office Version Excel 2016+ or Microsoft 365
Memory 4 GB RAM minimum
Disk Space 200 MB available
02

Download the Installer

Grab the latest version of Hackabacus for your platform.

Download the installer for Windows. The package includes the Excel and PowerPoint plugins.

Download for Windows
03

Windows Installation

Install Hackabacus on Windows 10 or later.

Windows
  1. Close all Microsoft Office applications (Excel, PowerPoint, Outlook, Word).
  2. Double-click the downloaded Hackabacus.MSI file. No need to Run as Administrator.
  3. If your work settings or Microsoft Defender SmartScreen prevent the application from starting as an unrecognized app, right-click the file, select Properties, and check Unblock under the General > Security section. Click Apply and OK. Repeat step 2.
  4. Follow the setup wizard: accept the license agreement, choose your installation directory (default is recommended), and click Install.
  5. Wait for the installation to complete. The wizard will register the COM add-in for Excel and PowerPoint automatically.
  6. Launch Excel or PowerPoint. You should see a Hackabacus tab in the ribbon. If prompted, click Enable to allow the add-in.
04

Activation & License Key

Activate Hackabacus with your license key to unlock all features.

  1. Open Excel and click the Hackabacus tab in the ribbon.
  2. Click the Activate License button in the toolbar.
  3. Enter your license key (sent to your email after purchase) and click Activate.
  4. A confirmation dialog will appear once activation succeeds. Your license is tied to your machine and Microsoft account.
  5. For team licenses, your IT administrator can deploy license keys via Group Policy or MDM. See the documentation for details.
05

Verify Installation

Confirm Hackabacus is properly installed and ready to use.

  • The Hackabacus tab is visible in the Excel ribbon toolbar.
  • The Hackabacus tab is visible in the PowerPoint ribbon toolbar.
  • Click Hackabacus > About and confirm the version number matches the latest release.
  • Run a quick test: select a cell range in Excel and apply a Hackabacus formatting preset to verify functionality.
  • Open PowerPoint and confirm the Hackabacus slide tools panel loads correctly.
06

Troubleshooting

Common issues and how to resolve them.

Plugin Not Showing in Ribbon

Go to File > Options > Add-ins. At the bottom, set the dropdown to COM Add-ins and click Go. Ensure Hackabacus is checked. If it is not listed, re-run the installer as Administrator.

COM Add-in Load Errors

If Excel reports that the add-in could not be loaded, first try restarting your computer. If the issue persists, open Programs and Features to repair the installation. Ensure your Office installation is up to date.

License Activation Issues

Verify that your license key is entered exactly as shown in your confirmation email (no extra spaces). Ensure your machine has an active internet connection during activation. If you have reached your device limit, deactivate a previous machine from your account dashboard or contact support@hackabacus.com.
